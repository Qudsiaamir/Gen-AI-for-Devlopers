# AI Learning Notes

This repository is a personal knowledge base for courses related to generative AI, software development, machine learning, and AI tools.

The structure is course-first so new courses can be added easily over time without changing the overall organization.

## Repository Structure

```text
.
├── README.md
└── courses
    ├── README.md
    └── deeplearning-ai
        └── generative-ai-for-software-development
            ├── README.md
            └── modules
                ├── module-01-introduction-to-generative-ai.md
                ├── module-02-pair-coding-with-an-llm.md
                └── module-03-leveraging-an-llm-for-code-analysis.md
```

## Organization Pattern

Each course should live inside:

```text
courses/<provider>/<course-name>/
```

Example:

```text
courses/deeplearning-ai/generative-ai-for-software-development/
courses/coursera/prompt-engineering-for-developers/
courses/fastai/practical-deep-learning/
```

## What Each Course Folder Can Contain

- `README.md` for the course overview
- `modules/` for module-wise notes
- `resources/` for links, PDFs, or references
- `labs/` for hands-on exercises
- `projects/` for assignments or final work
