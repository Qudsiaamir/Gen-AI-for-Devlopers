# Module 2: Pair-coding with an LLM

## Module Overview

This module focuses on how to use an LLM as a coding partner through prompting, feedback, role assignment, and tool-aware workflows.

## Learning Goals

- Use chatbots to generate code
- Improve results through iterative prompting
- Give effective feedback to an LLM
- Assign roles to improve output quality
- Understand coding tools powered by LLMs
- Learn context-window limits and best practices
